# Smart Document Processing System

## Overview
The Smart Document Processing System is a web-based application that automates the processing of educational documents. The system accepts documents in PDF, Word, and Excel formats and converts them into structured outputs.

## Features
- Supports PDF, Word, and Excel documents
- IQAC Module
- Committees Module
- Timetable Module
- Database Storage using SQLite
- Structured Output Generation
- User-Friendly Interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Pandas
- OpenPyXL

## Modules

### IQAC Module
Processes question papers and organizes questions based on Course Outcomes (COs) and Bloom's Taxonomy levels.

### Committees Module
Processes tabular documents such as committee lists and project team details.

### Timetable Module
Extracts timetable information such as day, time, subject, and faculty details.

## Workflow
1. Upload Document
2. Select Module
3. Process Data
4. Generate Structured Output
5. Store Data in Database

## Database
SQLite is used to store processed data for future retrieval.
